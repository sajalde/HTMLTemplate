PetroSoft

Petro Soft is specialized setup for Petrol Pumps. It Provides complete computerization of petrol pump with DIP reading, Pump Meter Reading, Auto Bill generation from Reading, Daily status chart summary, Lubricant Sales, Credit Sales, DIP stock v/s Actual stock, Density, Tank wise stock and lot more Petro Soft can handle multiple companies and multiple financial years. Petro Soft is GST ready and provides all required reports and calculation as per GST norms. Petro Soft is a Cloud based Software, also can be successfully installed in local LAN based or Individual computer.

Petro Soft Features

General Features

  • Create Multiple Organization & Multiple Financial years.
  • User can maintain the Meter Reading and enter the Testing & Handling/Evaporation Loss entries.
  • Vehicle Master - Petro Soft enables the users to store the party's multiple vehicles (very useful in recalling while credit sales entry).
  • Rate master Date wise & Location wise - This facility helps in storing the rates on different dates.
  • Item wise report is also available.
  • Credit Sales report is available in following manner: Party, Date, Memo and item wise (Detailed and Summary).
  • Party wise Credit Sales Report is available with its single vehicle or all vehicles.
  • This Software gives a Complete Track of Tanker Movement.
  • Petro Soft gives an option of maintaining stock at multiple locations.

User Control

  • Manage User (Add/ Edit /Delete User Profile)
  • Modify/ Delete / Add/ View rights
  • Fully Selected work control Access
  • User Login Time and IP Address Log
Pump Features
  • Pump Setup ( tbl_SYS_Client_CompanyInformation)
  • Branch Setup (tbl_SYS_Master_BranchInformation)
  • Shift (tbl_PS_Master_Shift)
  • FinancialYearSettings (tbl_PS_FinancialYearSettings)
  • GST Setup (tbl_PS_Master_GSTSetup)
  • Material Category (tbl_PS_Master_MaterialCategory)
  • UOM (Unit of Measurement) (tbl_PS_Master_UOM)
  • Item or Material Master (tbl_PS_Master_MaterialInformation)
  • Vendor Information (tbl_PS_Master_VendorInformation)
  • Customer Information (tbl_PS_Master_CustomerInformation)
Stock and Inventory Management
  • Purchase Lubricant with Details
  • Lubricant Sales – Direct/ Salesman wise
  • Delivery Challan Entry
  • Debit Bill Entry
  • Item wise/ Value wise Stock Report
  • Calculates Closing stock on As per Purchase Rate/Sale Rate
  • Location wise Stock Report
  • Salesman wise Stock Report
  • Item Ledger Report
  • Item Movement Report
